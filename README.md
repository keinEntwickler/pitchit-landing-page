# Pitchit Landing Page

Landing page for **Pitchit** — train your ear, one tone at a time.

Pitchit is an iOS ear-training game. Hear a tone, pick the note, and sharpen your musical ear one round at a time.

Live at: [getpitch.it](https://getpitch.it)

## Languages

- English (default, `/`)
- Deutsch (`/de/`)
- Italiano (`/it/`)
- Francais (`/fr/`)
- Espanol (`/es/`)
- Chinese (`/zh-Hans/`)

## Structure

- **Landing pages** — one per language, content from `_data/translations/`
- **Legal pages** — single multilingual documents (Privacy Policy, Legal Notice)
- **Open Source** — credits for Swift frameworks used in Pitchit

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Open [localhost:4000](http://localhost:4000/) in your browser.
